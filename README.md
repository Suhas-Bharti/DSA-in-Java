
### Arrays/Linear-Search
Contains problems solved using **Linear Search**:
1. `01_Find_Element_Index.java` - Find the index of an element.
2. `02_Check_Element_Exists.java` - Check if an element exists.
3. `03_Count_Occurrences.java` - Count occurrences of a key.
4. `04_Find_Max_Element.java` - Find maximum element.
5. `05_Find_Min_Element.java` - Find minimum element.

### Arrays/Binary-Search
Contains problems solved using **Binary Search**:
1. `01_Binary_Search_Basic.java` - Basic Binary Search.
2. `02_First_Occurrence.java` - Find first occurrence in sorted array.
3. `03_Last_Occurrence.java` - Find last occurrence in sorted array.
4. `04_Square_Root_Using_Binary_Search.java` - Find square root.
5. `05_Search_In_Rotated_Array.java` - Search in rotated sorted array.
